Never Miss a Meeting

Connect your Google Calendar and get popup reminders for your upcoming meetings. Snooze, dismiss, or join with a single click.

One-time setup takes about 3 minutes.

Step 1: Create a Google Project

Open Google Cloud Console, type a name like "My Calendar App" and click Create.

Project nameMy Calendar App
CREATE
Open Google Cloud Console

Step 2: Enable Calendar Access

Click the link below. You'll see a big blue "Enable" button — click it.

Google Calendar API
ENABLE
Open Calendar API Page

Step 3: Set Up Sign-In

A
OAuth consent screen — Choose "External", fill in your app name and email, then add your Gmail as a test user.
B
Create Credentials — "+ CREATE CREDENTIALS" → "OAuth client ID" → "Web application". Under "Authorized JavaScript origins" add: http://localhost:5500
C
Copy the Client ID — looks like 123456789-abc.apps.googleusercontent.com
OAuth Consent Screen Credentials Page

Final Step: Paste Your Client ID

Paste the Client ID below and click Connect. You'll only do this once.

After this, you'll never see this setup again.
